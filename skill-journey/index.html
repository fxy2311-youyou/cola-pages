✦ CHAPTER 05 ✦

🔤 字体迭代：最痛的调整

最初用Silkscreen做所有标题--像素感很强但中文完全看不清。经历了5轮迭代才找到平衡点:

V1 · 问题
Silkscreen全站

中文标题像素化后糊成一团,完全无法阅读 😵

V2 · 尝试
Space Mono 700 + Noto Sans SC 700

清晰了,但太厚重,放一起头重脚轻

V3 · 选字
5种中文复古字体对比

快乐体 / 黄油体 / 小薇体 / 毛笔体 / 落霞文楷 → 选了落霞文楷

V4 · 平衡
英文字体太粗,配不上文楷的轻盈

对比了 Outfit / DM Sans / Figtree / Space Mono 400

V5 · 定稿 ✓
Space Mono 400 + 字间距收紧

字重降下来,letter-spacing: -0.03em,终于和谐了 ✨
